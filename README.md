# doviz

> Get the newest exchange rates easily

## How to run
``` bash
# Clone the project
git clone https://github.com/serhatyuna/doviz.git

# Change the directory
cd doviz

# Before run the script, make sure you have installed libraries (BeautifulSoup, requests)
python doviz.py
```
